# Common Framework Architectures

A collection of different architectures based on modern frameworks (Vite, Elysia, Hono)

## Vite-Based Frontends

1. Single-page-application frontend vanilla TypeScript based on Vite with support of TSX: [vite-spa-plain-tsx](https://github.com/MansourM61/vite-spa-plain-tsx)
2. Multi-page-application frontend vanilla TypeScript based on Vite with support of TSX: [vite-mpa-plain-tsx](https://github.com/MansourM61/vite-mpa-plain-tsx)
3. Server-side-rendering single-page-application frontend vanilla TypeScript based on Vite (as middleware) with support of TSX and Express.js server: [middleware-vite-ssr-spa-express-plain-tsx](https://github.com/MansourM61/middleware-vite-ssr-spa-express-plain-tsx)
4. Server-side-rendering multiple-page-application frontend vanilla TypeScript based on Vite (as middleware) with support of TSX and Express.js server: [middleware-vite-ssr-mpa-express-plain-tsx](https://github.com/MansourM61/middleware-vite-ssr-mpa-express-plain-tsx/tree/main)

## Hono-Based Frameworks

1. Simple backend server with full support for TypeScript, and TSX: [hono-bun-plain-tsx](https://github.com/MansourM61/hono-bun-plain-tsx)
